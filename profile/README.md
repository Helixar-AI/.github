# Helixar

**Security infrastructure for the agentic AI era.**

As AI agents gain the ability to read your mail, write your code, and call your APIs, the attack surface expands in ways traditional security tools weren't built to handle. Helixar builds the protocols, tooling, and standards that keep agentic systems auditable, verifiable, and safe.

---

## What We Build

### [HDP — Human Delegation Provenance Protocol](https://github.com/Helixar-AI/HDP)
A cryptographic chain-of-custody standard for agentic AI systems. HDP gives every action taken by an AI agent a verifiable, tamper-evident trail back to the human who authorized it — making AI delegation auditable at runtime.

### [ReleaseGuard](https://github.com/Helixar-AI/ReleaseGuard)
An open-source artifact policy engine and hardening suite. ReleaseGuard scans, transforms, attests, and verifies release artifacts across build ecosystems — so the software you ship is exactly the software you built.

### [Sentinel](https://github.com/Helixar-AI/Sentinel)
An MCP security scanner. Sentinel helps teams evaluate the security posture of Model Context Protocol deployments before attackers do.

### [mcp-security-checklist](https://github.com/Helixar-AI/mcp-security-checklist)
A practical security baseline for engineers and platform teams deploying MCP solutions. Start here before you ship.

### [Unpinched](https://github.com/Helixar-AI/Unpinched)
A detection utility for PinchTab — a class of malware targeting AI-adjacent developer workflows.

## Focus Areas

- **Agentic AI security** — governance and runtime controls for AI agents operating with real-world permissions

- **MCP security** — hardening and scanning for Model Context Protocol infrastructure
- **AI threat detection** — tooling to identify and respond to emerging threats targeting AI-adjacent systems
