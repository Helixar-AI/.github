## Helixar Research Lab

Our open research arm. We publish protocols, tooling, and threat intelligence that we believe the entire AI security community should have access to.

### [HDP — Human Delegation Provenance Protocol](https://github.com/Helixar-AI/HDP)
A cryptographic chain-of-custody standard for agentic AI systems. Every action an agent takes carries a verifiable, tamper-evident trace back to the human who authorized it.

### [ReleaseGuard](https://github.com/Helixar-AI/ReleaseGuard)
An artifact policy engine and hardening suite. Scans, transforms, attests, and verifies release artifacts across build ecosystems.

### [Sentinel](https://github.com/Helixar-AI/Sentinel)
MCP security scanner. Evaluates the security posture of Model Context Protocol deployments.

### [mcp-security-checklist](https://github.com/Helixar-AI/mcp-security-checklist)
A security baseline for engineering and platform teams deploying MCP solutions.

### [Unpinched](https://github.com/Helixar-AI/Unpinched)
Detection utility for PinchTab — malware targeting AI-adjacent developer workflows.
